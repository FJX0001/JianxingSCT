# JianxingSCT
# BCC字幕转SRT格式转换工具

一个在线工具，用于将B站(Bilibili)的BCC字幕格式转换为标准的SRT字幕格式。

## 功能特性
- 支持.bcc、.json、.txt格式的BCC字幕文件
- 完全在浏览器本地处理，无需上传服务器
- 保留原始文件名
- 自动下载转换结果
- 响应式设计，支持移动设备

## 在线使用
访问：https://fjx0001.github.io/JianxingSCT/

## 本地运行
下载index.html
直接双击打开 `index.html` 文件即可在浏览器中运行。

## 技术栈
- HTML5
- CSS3
- Vanilla JavaScript
