# BCC字幕转SRT格式转换工具

一个基于Web的BCC字幕文件转换为SRT格式的工具，完全在浏览器本地处理，无需上传服务器，保障用户隐私安全。

## 🌟 项目特点

- **本地处理**：所有转换操作均在浏览器中完成，文件不会上传到任何服务器
- **批量处理**：支持一次性转换多个BCC字幕文件
- **格式支持**：兼容`.bcc`、`.json`、`.txt`格式的BCC字幕文件
- **保留文件名**：转换后自动保留原始文件名，仅扩展名更改为`.srt`
- **开源透明**：代码完全开源，可在GitHub上查看和贡献
- **用户友好**：简洁直观的界面，支持拖拽上传

## 📋 使用说明

### 单文件转换
1. 点击上传区域或拖拽单个BCC字幕文件到指定区域
2. 点击"转换字幕格式"按钮
3. 转换完成后，点击"下载SRT文件"保存结果

### 批量转换
1. 开启"批量处理模式"开关
2. 选择多个BCC字幕文件（支持拖拽）
3. 点击"转换字幕格式"开始批量处理
4. 转换完成后，点击"下载全部SRT文件 (ZIP)"获取压缩包

### 文件格式说明
- **BCC文件**：B站使用的字幕格式，通常以`.bcc`为扩展名
- **SRT文件**：通用字幕格式，可在大多数视频播放器和编辑软件中使用

## ⚠️ 安全提示

当下载批量转换的ZIP文件时，Windows系统可能会显示安全警告。这是正常的系统安全机制，可按以下步骤解决：

1. 右键点击下载的ZIP文件
2. 选择"属性"
3. 在常规选项卡底部，找到"安全"部分
4. 如果看到"此文件来自其他计算机..."的提示，请勾选"解除锁定"
5. 点击"确定"后即可正常解压

## 🚀 部署方式

本项目为纯静态网站，可通过以下方式部署：

### GitHub Pages（推荐）
1. Fork本仓库
2. 在仓库设置中启用GitHub Pages
3. 选择`main`分支作为源

### 本地部署
1. 克隆仓库到本地
2. 使用任何HTTP服务器（如Python的`http.server`、Node.js的`serve`等）启动
3. 浏览器访问本地服务器地址

## 🎯 支持的文件格式

### 输入格式
- `.bcc` - B站原生字幕格式
- `.json` - JSON格式的BCC字幕
- `.txt` - 包含BCC字幕内容的文本文件

### 输出格式
- `.srt` - 标准字幕格式

## ⌨️ 键盘快捷键

- `Ctrl + Enter`：执行转换
- `Ctrl + D`：下载单个SRT文件（单文件模式）
- `Ctrl + Shift + D`：下载批量ZIP文件（批量模式）
- `Esc`：清除所有内容
- `Ctrl + B`：切换批量模式
