# BCC字幕转SRT格式转换工具

一个在线工具，用于将B站(Bilibili)的BCC字幕格式转换为标准的SRT字幕格式。

## 在线访问

访问地址：https://fjx0001.github.io/JianxingSCT/

## 功能特性

- 🎯 **支持多种格式**：支持 .bcc、.json、.txt 格式的BCC字幕文件
- 🔒 **隐私保护**：完全在浏览器本地处理，无需上传服务器
- 📝 **保留文件名**：转换后自动保留原始文件名
- ⚡ **快速转换**：实时转换，即时下载
- 📱 **响应式设计**：支持桌面和移动设备
- ⌨️ **键盘快捷键**：
  - Ctrl + Enter：转换字幕
  - Ctrl + D：下载文件
  - Esc：清除内容

## 使用方法

1. **上传文件**：点击上传区域或拖拽BCC文件到指定区域
2. **格式转换**：点击"转换字幕格式"按钮
3. **下载结果**：点击"下载SRT文件"按钮保存转换结果
